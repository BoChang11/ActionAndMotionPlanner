## Prerequisites
- OPENGL library
- GLUT library
- BOOST library
- Pybullet library
- PDDL :fast-downward

## Getting Started
```
cmake -DCMAKE_BUILD_TYPE="Release"
make
./bin/Abetare GRunPlanner data/Prob3x3_Obj4_1.txt
```
- Press "r" until solved.
